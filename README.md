# forklift-desktop-project
# Реализованный функционал
-Плавный fade-in из чёрного в начале сцены
-Ипут система через свой Input Map
-Запуск двигателя
-Передвижение через Rigidbody
-Подъём и опускание вилки
-Груз и система появления-отправке
-Анимации груза при появляении-отправке
-Камера заднего вида, выведенная на экран в кабине
-Система топлива

# Дополнительно реализовано
-Весь интерфейс выполнен на UI Toolkit
-Меню паузы
-Отдельная сцена главного меню
-Визуальные маркеры точек появления и сдачи груза
-Архитектура на основе MVC и DI
-Внедрение зависимостей через Zenject
-Анимации реализованы с помощью DOTween
-Использованы кастомные skyboxы
-Использованы Cinemachine камеры
